Leitor de arquivo CSV

----------

Como executar
-------------

Para executar o programa você pode:

- Abrir o arquivo run.sh OU run.bat que encontram-se na pasta resource/dist.
- Executar o comando **java -Dfile.encoding=utf-8 -jar csv-reader.jar** pelo console ou prompt de comando, estando na pasta do arquivo csv-reader.jar.
- Executar a classe main pela IDE que preferir.
